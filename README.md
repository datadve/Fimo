<h1>FIMO</h1>
<h2>How to use?</h2>
<h3>1. Download or clone this repo.</br></h3>
<h3>2. Install dependencies.</br></h3>

```
npm install
// or
yarn install
```

<h3>3. Run project on iOS / Android.</h3>

```
 react-native run-android // npm run android
 // or
react-native run-ios // yarn android
```
